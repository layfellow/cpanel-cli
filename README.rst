===============================
cPanel 2 Command Line Interface
===============================

.. _README in English:

`LÉAME en español`_

A general CLI utility to run common tasks on a cPanel2-controlled web site.

Requirements
============

- Linux or macOS (it might work on Windows using WSL2 or MSYS2; don’t ask me about it)
- Python 3.8 or later

Installation
============

Run:

.. code:: sh

    $ pip3 install --user cpanel-cli

=============
